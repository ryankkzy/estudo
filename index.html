O que é JavaScript?
JavaScript é uma linguagem de programação usada principalmente para criar interatividade em páginas web.
O que é CSS?
CSS é uma linguagem de estilo usada para descrever a apresentação de um documento HTML.
O que é Flexbox?
Flexbox é um modelo de layout no CSS que permite a disposição flexível de itens dentro de um contêiner.
